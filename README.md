## Run Instructions

Prerequisites:

- [Node v16 or higher](https://nodejs.org/en)
- A phone (Android or IOS)
- The [Expo Go App](https://expo.dev/client)

1. Clone the repository

```bash
git clone <git@url>
cd react-native-hackpack
```

2. Install dependencies

```bash
npm i
```

2. Run

```bash
npm start
```
